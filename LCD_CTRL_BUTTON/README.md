# LCD Display Board

## 📂 Project Overview
This project demonstrates the design of an LCD display control board using Cadence Allegro.  
It highlights schematic capture, PCB layout, footprint creation, and Gerber generation for manufacturing.

## 🛠 Skills & Tools Used
- Schematic design (Cadence Allegro)  
- PCB layout workflows  
- Footprint creation & validation  
- Gerber generation for production  

## 📌 Files Included
- `LCD_CONTROL_BUTTON_CKT_D01R01.DSN` – Schematic file  
- `LCD_CONTROL_BUTTON_LIB.OLB` – Library file  
- `LCD_CONTROL_PCB_V01R08.brd` – PCB layout file  
- `GERBER IMAGES/` – Manufacturing outputs  


## 🎯 Goal
To design a manufacturable LCD control board and practice professional PCB workflows.
